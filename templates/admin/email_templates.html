About Email Templates: Customize the email messages sent to participants and faculty. You can use variables like {first_name}, {last_name}, {course_name}, and {rsvp_link} which will be automatically replaced with actual values.
Available Variables
Person Variables:
  • {first_name} - First name
  • {last_name} - Last name
  • {email} - Email address
  • {role} - Role (PARTICIPANT/FACULTY)
Course Variables:
  • {course_name} - Course name
  • {start_date} - Start date
  • {end_date} - End date
  • {location} - Location
Link Variables:
  • {rsvp_link} - RSVP form link
  • {info_link} - Info form link
  • {hotel_link} - Hotel request link
Hotel Variables:
  • {hotel_night1_label}
  • {hotel_night2_label}
  • {hotel_night3_label}
